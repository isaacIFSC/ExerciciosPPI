uma regras css é o que vai determinar o estilos dos meus elementos html, como por exemplo mudar cor, layout, tamanho dos elementos etc...


o css pode ser feito na linha do elemento, ou declarando a regra num arquivo css ou dentro da tag "style" e precisa de um seletor e a declaração da regra: fonte slide


um exemplo de regra css é essa tag p com cor diferente, ao professor acessar o codigo irá ver que usei um seletor de classe e mudei a cor do p para azul dentro da tag "style"
