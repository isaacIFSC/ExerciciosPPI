Seletores CSS

Seletores de Tipo (Tag)

Seleciona todos os elementos que correspondem a uma determinada tag HTML.

Exemplo: p { color: red; }

Seletores de Classe

Utilizados para selecionar elementos com uma classe específica.

Exemplo: .minha-classe { font-size: 16px; }

Seletores de ID

Seleciona um elemento com um ID específico.

Exemplo: #meu-id { border: 1px solid black; }

Seletores de Atributo

Selecionam elementos com base em seus atributos.

Exemplo: input[type="text"] { width: 100%; }

Seletores Descendentes

Seleciona todos os elementos que são descendentes de um elemento específico.

Exemplo: div p { color: green; }

Seletores Filhos

Seleciona todos os elementos que são filhos diretos de um elemento específico.

Exemplo: div > p { color: blue; }

Seletores Irmãos Adjacentes

Seleciona o próximo elemento irmão de um elemento específico.

Exemplo: h2 + p { margin-top: 0; }

Seletores Irmãos Gerais

Seleciona todos os elementos irmãos de um elemento específico.

Exemplo: h2 ~ p { margin-top: 0; }

Seletores de Pseudo-classes

Permitem estilizar elementos em um estado particular, como hover ou focus.

Exemplo: a:hover { text-decoration: underline; }

Seletores de Pseudo-elementos

Usados para estilizar partes de um elemento.

Exemplo: p::first-letter { font-size: 200%; }

Seletores Universais

Seleciona todos os elementos em um documento.

Exemplo: * { margin: 0; padding: 0; }

Seletores de Combinação

Combina diferentes seletores para criar regras mais específicas.

Exemplo: p.classe1#id1 { color: red; }
