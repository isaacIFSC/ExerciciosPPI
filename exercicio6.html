Podemos incluir o css de três formas: inline, internal, external

Inline: declarar um style na propria linha do elemento e colocar o css ali dentro. recomendado para dar estilo para um unico elemento!

Internal: declarar a tag style dentro da tag head e colocar o codigo css ali dentro. recomando utilizar para dar estilo para mais de um elemento.

External: Em um arquivo externo .css você coloca os códigos lá e faz o link do arquivo na tag head. recomendado para dar estilos globais que podem ser reutilizados
